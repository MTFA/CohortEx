About CohortEx Editor and StevensonCMS

CohortEx Parameters Editor lets CohortEx studies to be edited and parameterized directly on GitHub.

CohortEx Parameters Editor is based on StevensonCMS. StevensonCMS (name relates to Robert Louis Stevenson,
the author of "The Strange Case of Dr. Jekyll and Mr. Hyde" novel) is a micro-CMS leveraging GitHub for storing
the website content and the Jekyll templating engine for templating and site configuration.
